# sysexits.nim

sysexits.h porting for Nim

## Installation

```sh
$ nimble install https://github.com/sasaplus1/sysexits.nim
```

Or add to your `.nimble` file:

```nimble
requires "https://github.com/sasaplus1/sysexits.nim"
```

## Usage

```nim
import sysexits

echo ExOk # => 0
```
